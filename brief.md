## 4. STYLE VISUEL

### Direction Artistique

| Critère | Choix |
|---------|-------|
| **Tonalité** | Professionnelle, accessible, moderne |
| **Style** | Corporate sobre avec touches de modernité technologique |
| **Palette chromatique** | Bleu nuit (confiance), Blanc (clarté), touches d'orange/violet (innovation IA) |
| **Typographie** | Sans-serif moderne (ex: Inter, Montserrat) pour les titres + lisible pour le corps |
| **Élements graphiques** | Forms abstraites, icônes line, illustrations data-driven |

### Ambiances visuelles

- **Sobre** : Lignes épurées, beaucoup d'espaces blancs
- **Dynamique** : Micro-animations au scroll, transitions fluides
- **Corporate** : Photos professionnelles de dirigeants, environnements de travail
- **Tech** : Illustrations IA, data viz, code snippet décoratif

## 6. CONTENU PRINCIPAL À INTÉGRER

### Copy Global (ton à adopter)

- **Ton** : Expert mais accessible, orienté résultats, pas technobabla
- **Mapping** : Les décideurs ont peu de temps → messages clairs et impactants
- **Mots-clés IA** : Automatisation, productivité, ROI, transformation digitale
- **Mots-clés Stratégie** : Vision, croissance, organisation, gouvernance

### Contenus par page

#### Accueil
- **H1** : Votre prochaine transformation commence ici
- **Sous-titre** : Conseil stratégique et intelligence artificielle au service des dirigeants
- **CTA** : Découvrir comment transformer votre organisation

#### Services
- Pages détaillées avec :
  - Description du besoin client
  - Méthodologie
  - Livrables concrets
  - Cas d'usage
  - CTA vers contact

#### Réalisations
- **Format** : Challenge → Solution → Résultats (chiffres)
- **Types** : Entreprise +30%, automatisation processes, événementiel XX personnes

#### Blog
- Articles optimisés SEO (800-1500 mots)
- Formats variés : liste, étude de cas, interview, actu
- Call-to-action en fin d'article

## 7. CONTRAINTES TECHNIQUES

### Infrastructure

| Critère | Spécification |
|---------|---------------|
| **Hébergement** | Serveur français / RGPD (OVH, Scaleway, ou équivalent) |
| **Certificat SSL** | HTTPS obligatoire |
| **Framework** | Next.js, Nuxt, ou CMS type WordPress avec thème premium |
| **Base de données** | PostgreSQL ou MySQL |
| **CMS** | Optionnel - mais recommandation WordPress + Elementor Pro si client veut gérer |

### Performance

| Critère | Seuil |
|---------|-------|
| **Core Web Vitals** | LCP < 2.5s, FID < 100ms, CLS < 0.1 |
| **Score Lighthouse** | > 90/100 |
| **Mobile** | Chargement < 3s en 4G |
| **Images** | WebP, lazy loading, optimisation automatique |

### Accessibilité

- Conformité WCAG 2.1 niveau AA
- Navigation clavier
- Contraste des couleurs suffisant
- Balises ARIA pour les lecteurs d'écran

### Sécurité

- Protection DDoS
- Firewall applicatif
- Mises à jour régulières
- Sauvegardes quotidiennes
- Protection anti-spam (reCAPTCHA)

### SEO Technique

- Sitemap XML
- Robots.txt optimisé
- Schema.org (Organization, Service, Article)
- URLs nettoyées et sémantiques
- Balises canonical
- Open Graph et Twitter cards
